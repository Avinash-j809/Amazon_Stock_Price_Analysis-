# Amazon Stock Market Analysis

## Project Overview
This repository contains a comprehensive analysis of Amazon (AMZN) stock prices from its IPO in 1997 up to recent years. The analysis includes raw stock data, calculated summary statistics, pivot tables, and various visualizations that highlight trends, volatility, and trading volume patterns over time.

## Excel File Description

The main data file in this project is `Amazon-Stock-Analysis.xlsx`, which contains the following key components:

- **Amazon Stock Price Data**: Raw daily historical stock prices with fields such as open, high, low, close, adjusted close, and trading volume.
- **Pivot Tables & Charts**: Multiple sheets featuring pivot tables summarizing average prices and volumes by time frames like month, year, and day of the week.
- **Visualization Sheets**: Dedicated sheets containing charts that visualize price trends, volume patterns, and other key insights.
- **Dashboard**: An interactive sheet consolidating important visual summaries and key performance indicators for quick, at-a-glance analysis.

## How to Use

- Open the Excel file in Microsoft Excel or any compatible spreadsheet software.
- Review the raw data for granular day-to-day stock movements.
- Use pivot tables to explore aggregated statistics and trading patterns.
- Analyze visual charts for trend spotting and anomaly detection.
- Export the summarized data for further analysis in tools like Python, R, or Power BI if needed.

## Key Features

- Detailed multi-year stock price history.
- Statistical summaries by month, weekday, and year.
- Visualizations showing price trends and volume fluctuations.
- Dashboard for consolidated insights.
